This folder contains an example using netlink feature to communicate with the kernel.

- Build

``` shell
gcc -o main main.c
```

- Package source code

``` shell
tar -cf src.tar ./src
```

- Extract source code 


``` shell
tar -x -v -f ./src.tar
```
